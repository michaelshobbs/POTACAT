# Remote audio over Tailscale, multi-op logging, and a batch of fixes

**Headline:** v1.8.17 fixes remote-rig audio that went silent over **Tailscale**, adds **Ctrl/Cmd-click multi-op** spot selection and logging, **drag-to-reorder VFO profiles**, left-handed paddle support for the WinKeyer, and a stack of rig, JTCAT, and ECHOCAT fixes.

## 📡 Remote audio now works over Tailscale

When you used **Share My Rig** (or a remote-desktop client) over **Tailscale**, tuning and spots worked but you got **no audio in either direction**. The cause was a Chromium privacy feature that hides local IPs behind `.local` mDNS candidates — and those can never be resolved across a tailnet, so the only usable direct audio path (the `100.x` tailnet address) never connected. POTACAT now gathers real host candidates on every interface, so the audio link forms over Tailscale. LAN and Cloud-Tunnel (TURN relay) paths are unaffected. — thanks **W7RTA** and **N0SNO** for the reports

> Note: this fix is specifically the **Tailscale/LAN** direct path. Audio over **Cloud Tunnel** relies on the Cloudflare TURN relay; if you hit no-audio over Cloud, check that the shack is signed into Cloud with Cloud Tunnel enabled.

## 👥 Multi-op spot selection and logging

**Ctrl/Cmd-click** spots to select more than one, then log a **multi-op activation** in one go. Multi-selected rows now draw a full outline so the selection is obvious.

## 🎛️ VFO profiles & CW

- **Drag to reorder VFO profiles** in the Profiles tab.
- The **K1EL WinKeyer** now honors **paddle swap** for left-handed operators.

## 📝 Logging polish

- **Re-spot message presets** — pick a saved comment template when re-spotting after a QSO.
- Activation **banner falls back to cty.dat geo** when a spot has no coordinates.
- **LLOTA** reference examples and prefixes updated to the 4-letter (`LLCL-`) format.

## 🐞 Bug fixes

- **JTCAT:** double-clicking a station answering your CQ now correctly **advances and logs** the QSO.
- **JTCAT:** new option to keep **FT8/FT4 in plain USB** on older Yaesu rigs (e.g. FTDX3000) instead of forcing a data mode.
- **Spots:** the pinned (last-tuned) spot now honors your **band/mode/region filters**.
- **Scan:** hitting **skip on the active row** tunes the next spot immediately, and a **dead serial CAT** link now warns instead of silently stalling.
- **Rig:** generic **Icom** rigs now poll **TX power**, scaled per named CI-V model.
- **ECHOCAT:** the network-adapter filter now distinguishes **VPN overlays from hypervisor adapters** (keeps bridged Hyper-V working), and **mDNS no longer claims the Mac's own hostname** (which caused a rename on every launch).
- **Installer:** the Remote Launcher is now stopped **before** the app-running check during upgrade/uninstall.
- **Stability:** fixed a **dark-mode popout white flash** and **zombie processes / SIGBUS** on shutdown.
